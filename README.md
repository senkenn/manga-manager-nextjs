# Manga Manager

## System Architecture

- Web Framework: Next.js
  - App Router
  - Server Actions
- CSS Library: Tailwind CSS
- DB: DynamoDB
- Cron: EventBridge
- CI/CD: GitHub Actions
- IaC: AWS CDK

## Develop Tools

- Docker outside of Docker
- Package Manager: Yarn
- Validation: Zod
- State Management: Recoil
- Testing Tool: Vitest
